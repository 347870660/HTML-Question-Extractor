# HTML Question Extractor


超星学习通HTML题目提取工具，自动从HTML文件中提取选择题和主观题，生成结构化的文本文件。

## ✨ 功能特点

- 🚀 **自动识别**: 自动识别当前目录下的HTML文件
- 📝 **智能提取**: 支持选择题和主观题的智能提取
- 🎯 **格式优化**: 生成清晰易读的文本格式
- 🔧 **简单易用**: 一键处理，无需复杂配置
- 💾 **批量处理**: 支持多个HTML文件批量处理
